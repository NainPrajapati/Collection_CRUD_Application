# Collection CRUD Application

## 📌 Project Overview
This is a standard Java application designed to demonstrate the implementation of **CRUD operations** (Create, Read, Update, and Delete) using the **Java Collections Framework**. It provides a clean, logic-based approach to handling data without the need for an external database.

## 🚀 Features
*   **Create:** Add new records to a collection (e.g., ArrayList or HashMap).
*   **Read:** View all existing records or search for specific items by ID.
*   **Update:** Modify existing data entries.
*   **Delete:** Remove records from the collection securely.
*   **Search:** Efficiently find items using unique identifiers.

## 🛠️ Technologies Used
*   **Language:** Java (JDK 8 or higher)
*   **Framework:** Java Collections API
*   **IDE:** Eclipse

## 📂 Project Structure
*   `src/`: Contains the source code files.
*   `bin/`: Compiled bytecode.
*   `.project` & `.classpath`: Eclipse configuration files.

## ⚙️ How to Run
1.  **Clone the repository:**
    ```bash
    git clone https://github.com/NainPrajapati/Collection_CRUD_Application.git
    ```
2.  **Open in Eclipse:**
    *   File > Import > Existing Projects into Workspace.
    *   Select the cloned folder.
3.  **Run the App:**
    *   Locate the `MainApplication.java` file (or the file containing the `main` method).
    *   Right-click > Run As > Java Application.
